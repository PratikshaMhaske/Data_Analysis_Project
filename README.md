## üìå Project Overview
This project was completed as part of the **Data Analysis Internship at Cognifyz Technologies**.  
The objective of the project is to perform **exploratory data analysis (EDA)** on a restaurant dataset to extract meaningful insights related to cuisines, cities, pricing, services, ratings, votes, and geographic distribution.

The analysis is divided into **three levels**, each focusing on progressively advanced data analysis tasks.

## üìÇ Dataset Description
The dataset contains information about restaurants, including:
- Restaurant Name & ID
- City & Location details
- Cuisines offered
- Price range
- Online delivery & table booking availability
- Aggregate ratings & votes
- Geographic coordinates (Latitude & Longitude)

---

## üìä Analysis Levels and Tasks

### üîπ Level 1: Basic Exploratory Data Analysis
- Identified the **top 3 most common cuisines**
- Performed **city-wise analysis** to find:
  - City with the highest number of restaurants  
  - Average ratings by city
- Analyzed **price range distribution**
- Compared **online delivery availability** and its impact on ratings

---

### üîπ Level 2: Intermediate Analysis
- Identified **most common cuisine combinations**
- Performed **geographic analysis** using latitude and longitude
- Analyzed **restaurant chains**, their ratings, and popularity using votes

---

### üîπ Level 3: Advanced Analysis
- Analyzed restaurants with the **highest and lowest votes**
- Studied the **correlation between votes and aggregate ratings**
- Examined the relationship between **price range**, **online delivery**, and **table booking**
- Performed **rating-based analysis** in the absence of textual reviews

> ‚ö†Ô∏è Note: The dataset does not contain textual customer reviews. Therefore, sentiment analysis based on review text was not performed. Customer feedback was analyzed using ratings, rating text, and votes instead.

---

## üìà Key Insights
- Certain cuisines and cuisine combinations are significantly more popular.
- Restaurants are clustered in specific geographic regions, indicating urban concentration.
- Higher-priced restaurants are more likely to offer **table booking**, while lower-priced ones focus more on **online delivery**.
- A positive relationship exists between **votes and ratings**, showing that popular restaurants often receive better ratings.
